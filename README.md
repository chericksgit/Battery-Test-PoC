# Battery-Test-PoC
 Proof of Concept code for Battery Cell, Module, and Pack test

 Weld Resistance project was written in LabVIEW 2019. You can simulate the following hardware in MAX to run the LabVIEW example code.
 - NI 4071 DMM, device name: DMM
 - NI 4138 SMU, device name: SMU
 - NI 2530 Switch, device name: 2530B
 - NI 2575 Switch, device name: 2575
